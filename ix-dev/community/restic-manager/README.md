# Restic Manager

[Restic Manager](https://codeberg.org/echocat/truenas-restic-manager) is a TrueNAS-integrated backup manager that uses restic to back up selected TrueNAS datasets from snapshots.

It can pair with TrueNAS, discover datasets, create consistent snapshots for backups, run jobs on demand or on a schedule, and show backup run history with live progress.

Sign in to Restic Manager with your own TrueNAS user credentials. Currently, the TrueNAS user must have the `Full Admin` role for pairing and management operations.
